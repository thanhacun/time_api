API Basejump: Timestamp mircoservice !

User stories:

I can pass a string as a parameter, and I will check to see whether that string contains either a unix timestamp or natural language date (ex: March 24, 2016)

If it does, it returns both the Unix timestamp and the natural language form of that date

If it does not, it returns nulls

Exmaple usage:

https://thcn-time-api.herokuapp.com/March%2024,%202016
https://thcn-time-api.herokuapp.com/1450137600

Example output:

{ "unix": 1450137600, "natural": "December 15, 2015"}
